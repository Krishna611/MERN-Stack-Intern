# MERN Intern Assignment

## Backend Setup
```bash
cd backend
npm install
npm run dev
```
Runs on http://localhost:5000

Default admin:
- email: admin@example.com
- password: admin123

## Frontend Setup
```bash
cd frontend
npm install
npm start
```
Runs on http://localhost:3000

## Features
- JWT authentication
- Role-based access (Admin / Student)
- Admin: CRUD on students
- Student: view/update own profile
